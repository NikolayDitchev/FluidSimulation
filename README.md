# FluidSimulation
Fluid Simulation based on Jos Stam's paper Real-Time Fluid Dynamics for Games (http://graphics.cs.cmu.edu/nsp/course/15-464/Fall09/papers/StamFluidforGames.pdf)


The exe may crash on some versions of Windows 10. The dynamic-link library files must the in teh same folder with the exe to run it.

Controls:
  Left mouse button will add fluid to the container.
  Rigth mouse click will add solid to the container.
  
 Always when the mouse if moving in the container the surrounding fluid will react to it.
 The fluid interacts with the solid.
